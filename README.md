# AstroX S1 - Página Web

## Descripción

Este proyecto consiste en el desarrollo de una página web para promocionar un producto ficticio: los auriculares AstroX S1.

La página presenta información sobre el producto, sus características principales, información sobre la empresa, una sección de compra simulada y un formulario de contacto.

## Estructura del proyecto

* `index.html` → Página principal.
* `nosotros.html` → Información sobre la empresa.
* `contacto.html` → Formulario de contacto.
* `comprar.html` → Simulación de compra del producto.
* `style.css` → Estilos de la página.
* `boton.js` → Funcionalidades interactivas.
* `img/` → Imágenes e iconos utilizados.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Formspree (envío de formularios)

## Características

* Diseño responsive para PC y dispositivos móviles.
* Navegación entre múltiples páginas.
* Animaciones y efectos visuales.
* Formulario de contacto funcional.
* Página de compra simulada.
* Uso de imágenes y fondos personalizados.

## Objetivo

Aplicar conocimientos de desarrollo web utilizando HTML, CSS y JavaScript para crear una página funcional, organizada y visualmente atractiva.
